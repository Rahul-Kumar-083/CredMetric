# CredMetric | AI-Powered Financial Analytics Dashboard

CredMetric is a next-generation financial analytics platform designed for SMEs to visualize fiscal health through AI-driven insights. Built with **Next.js**, **Tailwind CSS**, and **Framer Motion**, it offers a premium, high-performance user experience with native dark/light mode support.

## 🚀 Key Features

- **AI Health Score**: Visual gauge representing overall financial health based on multi-factor analysis.
- **Explainable AI (XAI)**: Transparent breakdown of the factors (revenue growth, margins, etc.) impacting the financial score.
- **Modern Theme System**: 
  - Staged settings (changes apply only on Save).
  - High-contrast Light Mode optimized for professional accessibility.
  - Sleek Dark Mode with neon accents for high-tech visualization.
- **Risk Analysis**: Interactive donut charts segmenting financial, operational, and market risks.
- **Responsive Layout**: Fluid dashboard grid that handles small screens and sidebar states gracefully.
- **Secure Authentication**: Integrated auth guarding and session management.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Charts**: Recharts & Custom CSS Layouts
- **Icons**: Lucide React
- **State Management**: React Context API

## 📦 Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Rahul-Kumar-083/credmetric-frontend.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the dashboard.
